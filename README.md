# SimpleDisk
用tornado实现的远程文件管理系统（网盘）



## 安装tornado

```
pip3 install tornado
```



## 更改所管理的文件的根目录

分别位于 **config.py** 末行和**templates/files.html**的201行

请注意：**这两个目录必须相同**



## 运行

```shell
# git clone https://github.com/kainhuck/SimpleDisk.git
# cd SimpleDisk
# python3 ./server.py
```



## 访问

浏览器中输入 <你的IP地址>:8000
